让Vim 如虎添翼

用 IntelliJ 的时觉得 create gist 很好用，查了下，发现 vim 下也有这个插件，于是马上配置上。

安装

  1. 下载 Gist.vim

  2. 解压后进入目录，拷贝文件

  3. 如果你的 vim 没有安装 webapi-vim 还要把这个也安装上
    下载 webapi
    拷贝文件

  4. 设置 github 用户名

  5. 重启vim
    输入命令，会提示输入 github 的密码，确认后会把当前整个文档内容发送到 gist。
