检查安装ORACLE依赖的RPM软件包

oracle的安装所依赖的软件包，有一部分linux是没有安装的。这需要将这些尚未安装的软件包先安装后才能正常安装oracle。这些未安装的软件包在linux安装光盘上可以取到，我

[root@localhost oracle]#mount /dev/cdrom /mnt/cdrom
