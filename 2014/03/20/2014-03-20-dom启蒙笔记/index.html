dom 启蒙笔记

文档对象模型(Document Object Model),是个由javascript 节点对象组成的层次结构。
HTML 文档被浏览器解析并转换为一个由节点对象组成的以体现当前文档的树状结构。
DOM 的目的是使用Javascritp操作,为当前文档提供一个变成接口。
